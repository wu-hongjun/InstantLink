# Audit: 2026-05-19

## Scope

Audit before first implementation pass: scaffold assumptions, InstantLink protocol reference, and hard-coded Square Link constraints.

## Findings

- The scaffold was still Square-first in README, CLAUDE, ARCHITECTURE, ROADMAP, docs, and epics.
- InstantLink already supports four models: Mini, Mini Link 3, Square, and Wide.
- Model differences are not cosmetic: dimensions, chunk sizes, max JPEG size, packet delay, pre-execute delay, success code, and Mini Link 3 vertical flip differ.
- No application runtime existed before this pass; only placeholder package code was present.
- Hardware validation still cannot be performed on this machine. Camera USB-LAN, real BLE pairing, and real print tests remain target-device tasks.
