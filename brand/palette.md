# InstantLink Brand

The mark is an **Instax-style instant-photo frame** (thick bottom lip) framing a
**link** — two nodes joined by a bridge bar. It reads at 16 px and 1024 px, in
light and dark.

## Assets

| File | Use |
|------|-----|
| `instantlink-icon.svg` | Full-bleed app icon (1024²). iOS masks the corners — keep it square. |
| `instantlink-mark.svg` | Transparent-background mark for any surface. |
| `instantlink-logo.svg` | Horizontal mark + wordmark lockup. Outline the text to paths before external distribution. |

Rasterize with `cairosvg` (or `rsvg-convert`). App-icon PNG and the Bridge
boot splash are generated by `brand/render-assets.sh`.

## Palette

| Token | Hex | Use |
|-------|-----|-----|
| Teal (primary) | `#16B7A9` | Accent, links, active state, primary node |
| Teal deep | `#0B3A44` | Icon gradient base, photo-window fill, dark ink on light |
| Teal ink | `#0E3A43` | Wordmark on light, deepest brand text |
| Amber (secondary) | `#FFB020` | The linked/target node, highlights, warmth (Instax nod) |
| Paper | `#F6FBFA` | Film frame, light surfaces |
| Ink | `#0A1416` | Body text on light |

The teal gradient (`#17B6A7` → `#0B3A44`) evolves the Bridge's original
`#105060` boot-splash colour into a proper brand ramp.

## Type

Wordmark and UI: the platform system sans — **SF Pro** on Apple platforms,
**DejaVu Sans / Noto** on the Bridge LCD. Weight 700 for the wordmark with
`-4` tracking at display size; "Link" takes the teal accent.

## Usage notes

- Minimum mark size 16 px; the film-lip proportion is load-bearing — don't crop it.
- On dark surfaces use the mark as-is (the frame is light). On light surfaces the
  mark also works; add no extra container.
- Accent teal `#16B7A9` is the single interactive/brand colour across all three
  UIs (Bridge LCD, iOS, macOS). Amber is a sparing highlight, never a fill.
